 Task A - Вычисляет корень n-ой степени методом Ньютона и стандартными средствами.
    Метод Ньютона:
      1. Сделать начальное предположение x_0;
      2. Задать x_k+1 = x_k - f(x_k) / f'(x_k);
      3. Повторять шаг 2, пока не будет достигнута необходимая точность.
      https://en.wikipedia.org/wiki/Newton%27s_method
